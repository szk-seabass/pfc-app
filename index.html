💾 データ管理

📁 ファイルで保存・復元 (推奨)

スマホの「ファイル」アプリに直接保存・読み込みします。

📝 テキストで復元 (移行用)

古いアプリからコピーしたテキストがある場合は、ここに貼り付けて復元してください。

📖 使い方ガイド

📱 1. アプリをスマホに入れる

Safari(iPhone)またはChrome(Android)で開き、「共有ボタン」から「ホーム画面に追加」を選んでください。

🎯 2. 目標を自動計算する

「目標設定」エリアにプロフィール（年齢、身長、体重など）と日々の活動量を入力し、目的を選んで保存すると、最適なPFCバランスが自動で設定されます。

🍽 3. 食事を記録する

「＋ 新しい食事枠を追加」を押し、食材を登録します。手動や写真読取も可能です。

🔄 4. アプリが更新された時の対応

  1. 「💾 データ」からファイルをダウンロードしてバックアップを取る
  2. アプリをタスクキル（完全終了）する
  3. 開き直す（それでもダメなら再インストールしてファイルから復元）

ℹ️ アプリ詳細仕様

🧮 精密TDEE計算ロジック

本アプリは、単なる体重計算ではなく、ユーザーの「仕事の強度」と「運動の頻度」を個別に評価するハイブリッド方式を採用しています。

基礎代謝(BMR) = 10×体重 + 6.25×身長 - 5×年齢 + (性別補正)
総消費量(TDEE) = BMR × (仕事係数 + 運動ブースト)

⚖️ 目標PFCの算出アルゴリズム

導き出された目標カロリーに対し、ボディメイクに最も効果的とされる以下の比率で配分します。

P (タンパク質) = 体重 × 2.0g (筋合成を最大化)
F (脂質) = 体重 × 0.8g (ホルモンバランスを維持)
C (炭水化物) = 残りのカロリーを全て割り当て

📷 OCR写真解析エンジン

Googleのオープンソースエンジン「Tesseract.js」を使用。成分表の画像をアップスケールし、日本語特有の表記揺れ（たんぱく、蛋白質等）を自動補正・抽出します。

⚡ 技術スタック（PWA）

すべてのデータはサーバーに送られず、端末内（LocalStorage）に暗号化保存されます。これによりオフラインでも高速に動作します。
