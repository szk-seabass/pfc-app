ℹ️ アプリ詳細仕様

🧮 精密TDEE計算ロジック（v21）

本アプリは、単なる体重計算ではなく、ユーザーの「仕事の強度」と「運動の頻度」を個別に評価するハイブリッド方式を採用しています。

基礎代謝(BMR) = 10×体重 + 6.25×身長 - 5×年齢 + (性別補正)
総消費量(TDEE) = BMR × (仕事係数 + 運動ブースト)

【仕事係数（ベース）】

  • デスクワーク: 1.1
  • 立ち仕事・営業: 1.2
  • 重い肉体労働: 1.3

【運動ブースト（加算）】

  • ほぼしない: +0.0
  • 週1〜2回: +0.1
  • 週3〜5回: +0.25
  • 毎日・ハード: +0.4

⚖️ 目標PFCの算出アルゴリズム

導き出された目標カロリーに対し、ボディメイクに最も効果的とされる以下の比率で配分します。

P (タンパク質) = 体重 × 2.0g (筋合成を最大化)
F (脂質) = 体重 × 0.8g (ホルモンバランスを維持)
C (炭水化物) = 残りのカロリーを全て割り当て

これにより、筋肉を極力残しながら脂肪を落とす、科学的な栄養管理が可能になります。

📷 OCR写真解析エンジン

Googleのオープンソースエンジン「Tesseract.js」をブラウザ上で実行。撮影された成分表を200%に拡大し、グレイスケール化に近い処理を施してから文字を抽出します。また、日本語の表記揺れ（たんぱく、蛋白質、ﾀﾝﾊﾟｸ等）を正規表現で網羅的にスキャンします。

⚡ 技術スタック（PWA）

HTML5 / CSS3 / JavaScript(ES6+) を使用し、サーバーサイドを介さない完全なクライアントサイド実行を実現。ユーザーのプライバシー（食事内容やプロフィール）は外部に送信されず、端末内のLocalStorageに安全に暗号化（JSON形式）して保存されます。
