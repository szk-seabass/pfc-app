💾 データ管理

📁 ファイルで保存・復元 (推奨)

スマホの「ファイル」アプリに直接保存・読み込みします。

📝 テキストで復元 (移行用)

古いアプリからコピーしたテキストがある場合は、ここに貼り付けて復元してください。

📖 使い方ガイド

📱 1. アプリをスマホに入れる

Safari(iPhone)またはChrome(Android)で開き、「共有ボタン」から「ホーム画面に追加」を選んでください。

🎯 2. 目標を設定する

「目標設定」エリアに体重を入力し、目的を選んで保存します。

🍽 3. 食事を記録する

「＋ 新しい食事枠を追加」を押し、食材を登録します。手動や写真読取も可能です。

🔄 4. アプリが更新された時の対応

  1. 「💾 データ」からファイルをダウンロードしてバックアップを取る
  2. アプリをタスクキル（完全終了）する
  3. 開き直す（それでもダメなら再インストールしてファイルから復元）

ℹ️ アプリ仕様

🧮 PFC自動計算ロジック

体重と目標（減量・維持・増量）から、以下の計算式を用いて1日の目標値を自動算出します。

  • 総カロリー: 体重 × (減量31 / 維持35 / 増量40) kcal
  • P (タンパク質): 体重 × 2.0g
  • F (脂質): 体重 × 0.8g
  • C (炭水化物): 残りのカロリーから算出

📷 OCR写真読み取り機能

Tesseract.jsを利用し、パッケージの成分表写真から「たんぱく質」「脂質」「炭水化物」の数値をAIが自動抽出します。読み取りエラー防止のため、画像を2倍にアップスケールし、誤字（OやIなど）を自動補正するフィルターを内蔵しています。

🛠 マイ食材辞書機能

頻繁に食べる食材を独自のデータベースに登録可能です。「100g単位」のほか、おにぎりや卵などの「1個単位」での計算ロジックにも対応しており、柔軟な計量が可能です。

📊 カレンダー＆プログレス分析

目標に対する達成率をPFCごとのプログレスバー（ゲージ）で可視化。また、月間カレンダー画面では日々の総摂取カロリーを俯瞰し、中長期的なダイエット戦略の立案をサポートします。

⚡ オフライン＆高速動作（PWA）

サーバー通信を行わず、すべてのデータを端末内（LocalStorage）に保存するPWA（Progressive Web App）アーキテクチャを採用。圏外でもサクサク動き、入力ラグがありません。OSネイティブのシェア機能（Web Share API）によるテキスト共有や、JSONファイルによる安全なデータ移行に対応しています。
